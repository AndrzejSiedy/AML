AML Test App!

App created based on VS2017 Agular2 template ServerSide as per template is .NET Core 2.0

Please go to "Aml Service" tab to start/stop service
Short solution description:
It is basic, test application. Please check "Service" project folder to see settings, and AmlServiceWatcher class where all logic happens
